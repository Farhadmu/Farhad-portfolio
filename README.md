# Farhad-portfolio

Personal portfolio website showcasing my projects, skills, and experience as a Frontend Developer.

## 🌐 Live Site

[Visit Portfolio](https://your-netlify-url.netlify.app)

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Supabase (Contact form backend)

## 📂 Project Structure

```
farhad-portfolio/
├── index.html          # Main HTML file
├── portfolio.css       # Styles
```

## ✨ Features

- Responsive design (mobile, tablet, desktop)
- Dark theme with neon accents
- Animated skill cards with proficiency badges
- Dual-timeline education section
- Project cards with detailed modals
- Contact form (Supabase backend)
- Blog link (separate Next.js site)
